Muhammad Putra Subagja

Putra, yang baru saja ditugaskan sebagai Backend Developer, akan fokus pada bagian "belakang panggung" dari pembuatan situs web. Backend web development melibatkan pembuatan struktur dan logika yang mendukung fungsi-fungsi di situs web, tetapi tidak terlihat secara langsung oleh pengguna. Asep akan bertanggung jawab merancang, mengembangkan, dan memelihara server, database, dan berbagai komponen lain yang memastikan bahwa situs web berjalan dengan baik. Tugas-tugasnya termasuk mengelola data, menangani logika bisnis, dan memastikan interaksi yang lancar antara frontend dan backend sehingga pengguna dapat menikmati pengalaman online yang menyeluruh.

Deskripsi singkat tugas - Putra Subagja

Selama menjalani prakerin di PSTI FIK Universitas Bandar Lampung, tugas utama Putra sebagai backend developer adalah mengemban tanggung jawab dalam mengelola dan mengoptimalkan sistem. Dengan keahlian dan pemahamannya yang mendalam di bidang pengembangan backend, Putra bertanggung jawab untuk merancang, mengimplementasikan, dan memelihara fungsionalitas inti dari sistem. Tugasnya melibatkan pengembangan dan perbaikan modul backend, penyelarasan database, serta memastikan ketersediaan data yang akurat dan dapat diakses. Selain itu, Putra juga aktif berkolaborasi dengan tim pengembang dan departemen terkait untuk memastikan integrasi yang lancar antara berbagai komponen sistem. Dengan fokus pada peningkatan performa dan keamanan, Putra menjadi tulang punggung dalam menyusun solusi teknis yang efisien demi mendukung keberlanjutan operasional sistem.

Apa saja yang saya gunakan saat mengerjakan project ini?

1. Node JS

Node.js adalah lingkungan runtime JavaScript open-source yang memungkinkan pengembangan aplikasi backend efisien dan scalable. Berdasarkan mesin JavaScript V8 dari Google Chrome, Node.js menonjol dengan model non-blocking I/O, memungkinkan penanganan permintaan bersamaan tanpa memblokir eksekusi program. Keunggulan lainnya adalah manajemen paket yang kuat melalui npm, memudahkan pengembang mengakses ribuan paket dan modul JavaScript. Dengan dukungan asinkronus, kemampuan integrasi yang luas, serta komunitas yang besar, Node.js menjadi pilihan populer untuk membangun aplikasi server-side modern.

2. Git

Git adalah sistem kontrol versi distribusi yang populer, dirancang untuk melacak perubahan dalam kode sumber perangkat lunak selama pengembangan perangkat lunak. Dikembangkan oleh Linus Torvalds, Git memungkinkan tim pengembang untuk bekerja bersama secara kolaboratif dengan efisien dan menyelaraskan perubahan kode di berbagai komputer. Git menggunakan pendekatan distribusi, artinya setiap pengguna memiliki salinan penuh dari seluruh riwayat proyek. Ini memfasilitasi kolaborasi tanpa ketergantungan pada koneksi internet konstan. Git juga dikenal karena kecepatan dan efisiensinya dalam mengelola cabang (branching) dan penggabungan (merging), memungkinkan pengembang untuk eksperimen dengan fitur baru dan mengintegrasikan perubahan dengan mudah. Git, sering digunakan bersama dengan platform hosting seperti GitHub, GitLab, atau Bitbucket, telah menjadi alat kritis dalam pengembangan perangkat lunak modern, memberikan kendali yang kuat dan transparan terhadap evolusi kode sumber.

3. MongoDB

MongoDB adalah sistem manajemen basis data (DBMS) yang bersifat open-source dan berbasis dokumen. Dikembangkan oleh perusahaan MongoDB Inc., MongoDB menggunakan format data yang dikenal sebagai BSON (Binary JSON) untuk menyimpan dan mengelola data. Keunikan MongoDB terletak pada model dokumen yang fleksibel, di mana data disimpan dalam format JSON-like yang dapat mencakup struktur yang kompleks. Ini memungkinkan pengembang untuk menyimpan dan mengakses data dengan cara yang lebih alami dan mudah dimengerti.
