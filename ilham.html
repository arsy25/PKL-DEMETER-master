Deskripsi singkat tugas - Ilham Alparsy

Selama menjalani prakerin di PSTI FIK Universitas Bandar Lampung, tugas utama saya adalah sebagai frontend developer yang memanfaatkan Quasar Vue.js untuk mengembangkan tampilan web. Dalam peran ini, saya bertanggung jawab merancang dan mengimplementasikan antarmuka pengguna yang responsif dan user-friendly untuk mendukung penggunaan sistem secara optimal. Selain itu, saya juga terlibat dalam pengembangan alat IoT RFID dengan menggunakan RabbitMQ sebagai servernya. Tujuan utama dari proyek ini adalah mencapai fitur cashless untuk sistem absensi dan top-up. Dengan menggabungkan teknologi Quasar Vue.js untuk frontend dan RabbitMQ untuk manajemen server pada alat IoT RFID, saya berkontribusi pada upaya menciptakan solusi yang efisien dan modern dalam mendukung tujuan cashless serta meningkatkan pengalaman pengguna secara keseluruhan.

Apa saja yang saya gunakan saat mengerjakan project ini?

1. Quasar Dev

Quasar Dev digunakan sebagai lingkungan pengembangan untuk proyek frontend dengan Vue.js. Alat ini memudahkan proses pengembangan, pengujian, dan penyesuaian antarmuka pengguna. Dengan dukungan fitur hot-reloading, debugging, dan kemampuan linting yang baik, Quasar Dev mempercepat siklus pengembangan frontend.

2. RabbitMQ

RabbitMQ berperan sebagai sistem manajemen pesan yang efisien dalam proyek ini. Sebagai server pesan berbasis protokol AMQP, RabbitMQ mendukung komunikasi asinkron antara perangkat keras dan perangkat lunak. Penggunaan RabbitMQ memungkinkan alat IoT RFID untuk berkomunikasi secara efektif dengan sistem lain, seperti frontend, untuk mencapai fitur cashless absensi dan topup.

3. Arduino IDE

Arduino IDE digunakan sebagai lingkungan pengembangan untuk pemrograman perangkat keras menggunakan platform Arduino. Melalui Arduino IDE, saya dapat mengembangkan kode untuk kontrol dan interaksi dengan modul RFID pada alat IoT. Selain itu, IDE ini menyederhanakan proses pemrograman dan pengunggahan kode ke perangkat keras, memfasilitasi integrasi antara perangkat keras dan sistem secara keseluruhan.
